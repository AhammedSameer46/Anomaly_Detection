# Anomaly Detection using Machine Learning

## Overview
This project implements an anomaly detection pipeline to identify outliers in structured datasets using machine learning. It includes data preprocessing, feature scaling, model training, and threshold optimization to improve F1-score performance.

## Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter
